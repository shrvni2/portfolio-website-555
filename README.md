# 📄 Portfolio Website
A modern and fully responsive portfolio website built using JavaScript, HTML, and CSS.

## Description
This project is a fully responsive portfolio website, designed to showcase an individual's work and skills in a clean and modern way. The website is built using JavaScript, HTML, and CSS, and is optimized for use on a variety of devices.

## Features
* Fully responsive design
* Modern and clean user interface
* Easy to navigate and use
* Compatible with a variety of devices and browsers
* Customizable to fit individual needs and styles

## Tech Stack
* JavaScript (ES6+)
* HTML5
* CSS3
* Parcel (bundler)
* Jest (testing framework)

## Installation Steps
1. Clone the repository: `git clone https://github.com/shrvni2/portfolio-website.git`
2. Install dependencies: `npm install`
3. Start the development server: `npm start`
4. Build the project: `npm run build`
5. Run tests: `npm test`

## Usage
To use this project, simply open the `index.html` file in a web browser. The website is fully responsive, so it can be viewed on a variety of devices.

### Code Examples
```javascript
// script.js
import { Component } from './components.js';

const component = new Component();
component.render();
```


## Project Structure
* `index.html`: The entry point of the application
* `script.js`: The main JavaScript file
* `style.css`: The main CSS file
* `components.js`: A module containing reusable UI components
* `utils.js`: A module containing utility functions
* `tests/main.test.js`: A test file for the application
* `package.json`: The project's package file
